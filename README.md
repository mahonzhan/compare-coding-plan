# awesome-coding-plan
各厂家 Coding Plan 实际价值对比

**周期说明**：5h 五小时，w 周，m 月

**倍率说明**：额度倍率=当前周期额度/包月价格

### 关键结论
- 个人玩龙虾买 MiniMax Coding Plan Plus ￥49 最实惠，额度限制最小，倍率最高，且支持 TTS 和 图像生成，是唯一一家支持全模态、没有周限制的 Coding Plan
- 团队使用推荐购买 Kimi Code Allegretto ￥199 套餐，效果更好，一般强度下基本够用，价格合适，不够可以升级到 ￥699
