## 📖 术语说明

- **周期**：`5h` = 5小时 | `w` = 周 | `m` = 月
- **额度倍率**：`当前周期额度 ÷ 包月价格`（倍率越高，代表性价比越高）

---

## 💡 核心选购建议

- 🏆 **个人养龙虾性价比首选：`MiniMax Coding Plan Plus` (￥49/月)**
  - **优势**：目前最实惠、额度限制最小、倍率最高，唯一支持全模态（包含 TTS 和图像生成），非重度用户 ￥29 Starter 1/2.5 Plus 额度估计也够用

  - **特点**：模型代码能力更优，较高强度的日常开发够用，但倍率较差。
  - **扩展**：如果团队使用量增加，可多买几个 ￥199 套餐，或升级至 ￥699 套餐（不如买 3 个 ￥199 套餐划算）
